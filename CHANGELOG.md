# Changelog

Release notes for The Universe Decides, in every language the app supports.
Each entry is written once here and reused verbatim for both the Google Play
"What's new" listing and the corresponding GitHub Release description.

- List Draw now has a spinning wheel mode — a more visual way to let chance pick from your options. 🎡
